# callan-jackson.github.io

My personal portfolio site — live at **[callan-jackson.github.io](https://callan-jackson.github.io)**.

Vanilla HTML, CSS & JS. No framework, no build step, no dependencies — clone it and open `index.html`.

## What's in here

- `index.html` — the single-page portfolio (hero, about, skills, selected work, contact)
- `styles.css` — all styling, including the mobile nav
- `main.js` — canvas flight-path animation, scroll reveals, animated counters, mobile menu
- `Callan_Jackson_CV.html` — my CV, print-styled for A4 (open it and hit ⌘P / Ctrl+P to save as PDF)

## Notes

- Deployed automatically by GitHub Pages from `main`.
- Respects `prefers-reduced-motion` — animations are disabled for users who ask for that.
- The hero animation is a hand-rolled `<canvas>` starfield with quadratic-Bézier flight arcs — a nod to my previous life as easyJet cabin crew.
